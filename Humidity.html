Humidity

From our list of 500 cities, we query the OpenWeatherMap API for the humidity along with the latitude. The responses are taken from JSON and appended onto list. Next we create a dataframe to hold these list and use MatPlotLib to generate a scatter plot of latitude vs. humidity.

Looking at the visualization, there are no trends that are easily revealed. One interesting thing to note is that there are cities with 100% humidity at all lattitudes.
